# Sidebar Features Removal

## Removed Features (March 2026)

The following features were temporarily removed from the sidebar to simplify the UI and fix core chat functionality issues:

1. **System Status** - Real-time GPU/VRAM/CPU metrics display
2. **Common Questions** - Dropdown for quick-access veteran questions
3. **Models** - List of installed Ollama models
4. **Knowledge Base** - Page counts by source
5. **VRAM Warnings** - High VRAM usage alerts

## Reason

Chat functionality issues (duplicate question detection, message display timing) needed focused debugging without sidebar complexity.

## Future

These features will be re-added once core chat functionality is stable.
